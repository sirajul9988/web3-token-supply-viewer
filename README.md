# Web3 Token Supply Viewer

This is a simple Web3 project.  
It helps you check the total supply of a crypto token on a blockchain network.

This project is made for beginners who want to learn how tokens and smart contracts work.

## What This Project Does
- Read token contract address
- Fetch total token supply
- Show supply in console
- Save token data to file

## Tools Used
- Node.js
- Simple JavaScript
- Public blockchain API (mock)
- Local JSON file for storage

## How To Install
1. Install Node.js
2. Download this repository
3. Open terminal in project folder
4. Run:
   npm install
5. Start the app:
   node app.js

## Folder Structure
- app.js = main file
- config.js = project settings
- token.js = token logic
- api.js = API calls
- saver.js = file saving
- data/ = saved token data

## Example Use
- Set token contract address
- App shows total supply
- App saves token info
